🎬 CineLux Vadodara — Premium Movie Seat Booking
A modern, elegant, and fully interactive movie seat booking system built using HTML, CSS, and JavaScript.
This project offers a premium UI experience inspired by real-world cinema booking apps (PVR, INOX, BookMyShow).

✨ Features
🎞 Gujarati movie listings with Indian pricing (₹)
💺 Interactive seat selection (Regular, Premium, VIP)
🔒 Reserved seats system
🪑 Realistic theatre layout with aisle spacing
💰 Live price calculation (INR)
💾 LocalStorage saving (keeps selection even after refresh)
🌐 Responsive, world-class UI
📍 Theatre location: Vadodara, Gujarat

🚀 Tech Used
HTML5
CSS3 (custom premium UI)
Vanilla JavaScript
No frameworks needed — runs in any browser.

💡 How to Use
Clone or download the repository
Open index.html in your browser
Select your Gujarati movie
Choose seats and confirm your booking

🏆 Perfect For
Beginners learning JS DOM
UI/UX practice
College projects
Cinema booking system demos
